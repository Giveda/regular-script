# 明心版四书

## 简介
本项目包含《明心版大学》和《明心版中庸》两部作品，是对儒家经典文献的特别整理版本。这些文本旨在以一种新的方式呈现古代智慧，便于现代读者理解和学习。

## 内容结构
- **《明心版大学》**：包括前言、致谢、正文（分章节）、凡例及后记。
- **《明心版中庸》**：同样包含前言、致谢、正文（分章节）、凡例及后记。
